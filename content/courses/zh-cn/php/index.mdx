# PHP 简介

PHP（PHP: Hypertext Preprocessor）是一种广泛使用的开源服务器端脚本语言，特别适合于 Web 开发。它可以嵌入到 HTML 中，与数据库交互，处理表单，创建动态页面内容等。

## 为什么选择 PHP？

1. **易学易用**：PHP 的语法简单直观，对初学者非常友好
2. **广泛应用**：许多知名网站如 Facebook、WordPress 都使用 PHP
3. **丰富的生态系统**：大量的框架和库可供选择
4. **优秀的文档**：完善的官方文档和活跃的社区支持

## 开发环境搭建

要开始 PHP 开发，你需要：

1. **Web 服务器**（如 Apache 或 Nginx）
2. **PHP 解释器**
3. **数据库**（如 MySQL）

### 使用集成环境包

推荐使用以下集成环境包快速搭建开发环境：

- Windows：XAMPP, WampServer
- macOS：MAMP
- 跨平台：Laravel Homestead（基于 Vagrant）

## 第一个 PHP 程序

创建一个 `hello.php` 文件：

```php
<?php
  echo "Hello, World!";
?>
```

将文件放在 Web 服务器目录下，通过浏览器访问即可看到输出。

## 学习路线

1. PHP 基础语法
   - 变量和数据类型
   - 运算符
   - 控制结构

2. 函数和数组
   - 函数定义和调用
   - 数组操作
   - 字符串处理

3. 面向对象编程
   - 类和对象
   - 继承和多态
   - 接口和抽象类

4. 数据库操作
   - MySQL 基础
   - PDO 使用
   - 查询构建器

5. Web 开发实践
   - 表单处理
   - 会话管理
   - 文件操作
   - 安全性考虑
