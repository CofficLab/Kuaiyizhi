## 项目介绍

在这个系列教程中，我们将从零开始开发一个功能丰富的在线工具箱网站。通过实践，你将学习到现代前端开发的各种技术和最佳实践。

## 技术栈

- **前端框架**：Nuxt.js 3.x
- **UI 框架**：
  - TailwindCSS：原子化 CSS 框架
  - DaisyUI：基于 TailwindCSS 的组件库
- **API 集成**：聚合数据开放 API
- **部署**：Netlify
- **开发工具**：
  - VS Code
  - Git
  - Node.js

## 项目架构

```bash
project/
├── components/        # 可复用组件
├── pages/            # 页面组件
├── layouts/          # 布局组件
├── composables/      # 组合式函数
├── stores/           # 状态管理
├── utils/            # 工具函数
├── types/            # TypeScript 类型定义
├── public/           # 静态资源
└── server/           # API 路由
```

## 准备工作

开始学习之前，请确保：

1. 安装 Node.js（版本 >= 16）
2. 熟悉 JavaScript/TypeScript 基础
3. 了解 Vue.js/Nuxt.js 基本概念
4. 准备好开发环境：
   - 安装 VS Code
   - 安装 Git
   - 注册 GitHub 账号
   - 注册聚合数据账号

## API 准备

1. 注册聚合数据账号：[聚合数据](https://www.juhe.cn/)
2. 申请以下API：
   - 天气预报API
   - 全国快递物流查询API
   - 汇率换算API

## 注意事项

1. API 密钥安全
   - 不要直接在代码中暴露 API 密钥
   - 使用环境变量管理敏感信息
2. 开发规范
   - 遵循 ESLint 规则
   - 编写清晰的注释
   - 保持代码整洁
3. 版本控制
   - 定期提交代码
   - 编写清晰的提交信息
   - 创建功能分支
