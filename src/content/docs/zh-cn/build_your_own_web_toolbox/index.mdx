在这个系列教程中，我们将从零开始开发一个功能丰富的在线工具箱网站。

## 技术栈

- **编程语言**：[TypeScript](https://www.typescriptlang.org/)
- **前端框架**：[Astro](https://astro.build/)：现代化的 Web 框架，支持静态生成和服务端渲染
- **UI 框架**：
  - [TailwindCSS](https://tailwindcss.com/)：原子化 CSS 框架
  - [DaisyUI](https://daisyui.com/)：基于 TailwindCSS 的组件库
- **API 供应商**：[聚合数据](https://www.juhe.cn/)
- **部署**：[Netlify](https://www.netlify.com/)
- **开发工具**：
  - [VS Code](https://code.visualstudio.com/)
  - [Git](https://git-scm.com/)
  - [Node.js](https://nodejs.org/)

## 功能模块

1. 天气查询
   - 城市天气查询
   - 天气预报展示（未来7天）
   - 实时天气更新
   - 响应式设计

2. 快递查询
   - 快递单号查询
   - 物流信息展示
   - 查询历史记录
   - 常用快递公司收藏

3. 更多工具
   - 汇率转换
     - 实时汇率查询
     - 多币种支持
     - 历史汇率走势
   - 二维码生成
     - 文本/链接转二维码
     - 自定义二维码样式
     - 二维码导出
   - 图片处理
     - 图片压缩
     - 格式转换
     - 简单编辑

## 准备工作

开始学习之前，请确保：

1. 安装 Node.js（版本 >= 16）
2. 熟悉 JavaScript/TypeScript 基础
3. 了解 Astro 基本概念
4. 准备好开发环境：
   - 安装 VS Code
   - 安装 Git
   - 注册 GitHub 账号
   - 注册聚合数据账号

## API 准备

1. 注册聚合数据账号：[聚合数据](https://www.juhe.cn/)
2. 申请以下API：
   - [天气预报API](https://www.juhe.cn/docs/api/id/109)
   - [全国快递物流查询API](https://www.juhe.cn/docs/api/id/106)
   - [汇率换算API](https://www.juhe.cn/docs/api/id/107)

## 注意事项

1. API 密钥安全
   - 不要直接在代码中暴露 API 密钥
   - 使用环境变量管理敏感信息
2. 开发规范
   - 遵循 ESLint 规则
   - 编写清晰的注释
   - 保持代码整洁
3. 版本控制
   - 定期提交代码
   - 编写清晰的提交信息
   - 创建功能分支
4. <span class="text-yellow-500">⚠️ 中国大陆用户需要保证自己的网络能访问 GitHub</span>

让我们开始构建属于你的在线工具箱吧！
